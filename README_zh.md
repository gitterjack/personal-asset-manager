# Alive (活着) - 个人资产管理平台 📱💰

> **“无感记录，财务洞察，数字遗产。”**

Alive 是一个全面的个人资产管理平台，旨在帮助您轻松追踪所拥有的每一件物品，了解消费的真实成本，并管理您的数字遗产。

---

## 🌟 核心理念

我们认为资产管理不应成为一项负担。Alive 专注于：
- **无感记录**：快速记录实物资产和数字资产。
- **财务洞察**：超越简单的价格标签，理解**每日成本 (CPD)** 和市场趋势。
- **数字遗产**：确保您的数字足迹和资产能够安全且有意愿地传承。

---

## 🛠 关键模块

### 📊 资产管理 (Asset Manager)
- **动态仪表板**：实时汇总您的资产总数和净值。
- **每日成本 (CPD)**：自动计算您的电子产品在持有期间每天的真实成本。
- **市场统计（智能见解）**：获取百分位排名，将您的购买价格与其他用户进行对比。
- **转售指南**：集成 **闲鱼 (Goofish)** 进行快速估值。特色功能 **“iPhone 8个月法则”** —— 在每年贬值高峰前发出智能提醒。

### 🤝 社区
- **保值与跳水榜单**：发现哪些产品是“理财产品”（如 Fujifilm X100VI），哪些是“跳水冠军”（如三星 Galaxy S 系列）。
- **用户动态**：在社区分享您的“败家日记”或庆祝“成功回血”。
- **年度报告**：生成个性化的“2026 年度消费与投资报告”，可视化您的年度支出习惯。

### 🔐 数字资产与遗产
- **保险库 (The Vault)**：安全存储订阅账户（如 Netflix）、加密钱包（如 BTC）和保单。
- **隐私硬盘**：支持本地加密硬件连接。您的敏感数据保留在*您的*硬盘上，而非我们的云端。
- **数字遗嘱与遗产**：创建“数字墓志铭”或私人遗嘱。通过“30天沉寂”机制触发，您的资产可自动共享给指定的受益人。

### 👤 个人资料与网络
- **个人信息**：追踪您在不同人生阶段的财务旅程。
- **家族树**：与家人连接，管理共有资产和遗产计划。
- **净值汇总**：整合实物、数字和财务状况的统一视图。

### 💎 商业模式
- **零费率订阅**：革命性的“利息抵扣”模式。通过在合作伙伴基金（如余额宝）中保持余额，产生的利息自动覆盖您的月度订阅费。
- **透明化变现**：清晰的佣金信息和极简、非侵入式的广告。

## 🚀 技术栈

- **前端**：
  - [React 19](https://react.dev/) - 现代 UI 组件。
  - [Tailwind CSS 4](https://tailwindcss.com/) - 原子化样式。
  - [Lucide Icons](https://lucide.dev/) - 精美且一致的图标库。
  - [Vite](https://vitejs.dev/) - 极速构建工具。
- **后端**：
  - [Node.js](https://nodejs.org/) 与 [Express](https://expressjs.com/) - 稳健的 API 层。

- **AI 集成**：
  - [阿里云 DashScope (通义万相-v1)](https://help.aliyun.com/product/271257.html) - 根据您的资产画像生成定制的 3D IP 头像。

## 💻 安装指南

### 前提条件
- Node.js (v18 或更高版本)
- npm 或 yarn

### 1. 后端设置
```bash
cd server
npm install
node index.js
```
服务器将运行在 `http://localhost:3002`。

### 2. 前端设置
```bash
cd client
npm install
npm run dev
```
打开浏览器并导航至 Vite 提供的 URL（通常为 `http://localhost:5173`）。

### 3. Docker (生产环境)
您也可以使用 Docker 运行整个项目：
```bash
docker build -t alive-app .
docker run -p 3001:3001 -v $(pwd)/data:/app/server/data alive-app
```
在生产模式下，应用将通过 `http://localhost:3001` 提供服务。
